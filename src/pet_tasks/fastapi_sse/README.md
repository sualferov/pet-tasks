# Serve-Side Events

<p>
Решил попробовать Server-Side Events в связке с FastAPI & Redis PUB/SUB.<br>
</p>

<p>Что реализовал в рамках этого задания:</p>
<ul>
    <li>API /sse/stream - для получения потока данных</li>
    <li>Каждый клиент подписывается на уведомления от разных источников</li>
    <li>При публикации события, каждый клиент получает свой набор данных</li>
    <li>Подключил использование Redis</li>
    <li>API /sse/publish - для публикации события</li>
</ul>
